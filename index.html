Information

Memory is a counter game where the object is to find pairs. When the game begin, you may see cards for two seconds, then they will be hidden.

Rules

  • Select two cards to try to match the pictures
  • If you match the pictures you can go again
  • If they don't match it is the computer turn them
  • The player that find all pairs wins
  • Have fun!!!
